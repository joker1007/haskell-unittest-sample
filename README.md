# Haskell UnitTest Sample

## Usage

```sh
% cabal install --enable-test --only-dependencies

% cabal configure --enable-tests
% cabal build
% cabal test

# or

% cabal-dev install --enable-test

% cabal-dev configure --enable-tests
% cabal-dev build
% cabal-dev test
```
